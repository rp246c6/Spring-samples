## 🚀 Getting Started

### Prerequisites
- **Java 17+** (or your preferred version compatible with Spring)
- **Maven** or **Gradle** for build management
- **MySQL/PostgreSQL** (if running database-related samples)
- **Spring Boot** (for application demos)

### Clone the Repository
```bash
git clone https://github.com/rp246c6/Spring-samples.git
cd Spring-samples
```

### Build & Run
If using Maven:
```bash
mvn clean install
mvn spring-boot:run
```

If using Gradle:
```bash
./gradlew build
./gradlew bootRun
```
